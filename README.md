# Ahmad Shapan - "EL-KASHIF" Cinematic Portfolio

A high-end, editorial-style technical portfolio showcasing custom WordPress architectures and system engineering projects.

## Visual Identity (Al-Kashif)
- **Palette**: Black/Gold/Red cinematic theme.
- **Typography**: Plus Jakarta Sans for a premium feel.
- **Grid System**: 12-column investigative dashboard architecture.

## Key Features
- GSAP-powered scroll animations and interactive custom cursor.
- Performance-optimized project lightbox.
- Technical case study breakdowns (Challenge, Solution, Impact).

## Technologies
- HTML5 / CSS3 / Tailwind CSS
- GSAP (GreenSock Animation Platform)
- FontAwesome 6.4
